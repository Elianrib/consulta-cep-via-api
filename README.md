# Consulta de CEP via API (ViaCEP)

Este projeto é um formulário de endereço inteligente que permite ao usuário digitar um CEP e preencher automaticamente os campos de Logradouro, Bairro, Cidade e UF usando a API pública do ViaCEP.

## Funcionalidades
- Digitar um CEP e buscar endereço automaticamente.
- Formatação automática do CEP enquanto digita (XXXXX-XXX).
- Mensagem de feedback caso o CEP seja inválido ou não encontrado.
- Botão para buscar e suporte ao pressionar ENTER no campo CEP.

## Tecnologias utilizadas
- HTML
- JavaScript (fetch API e ES6+)
